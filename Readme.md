# Form Structure Drools

Library to represent a form after ABCD drools rules has been executed on it. It hosts any score and variable that has been created by the drools rules. 


# Usage

This is a Jar library that can be imported in any maven project as follows:

```
<dependency>
    <groupId>com.biit-solutions</groupId>
    <artifactId>form-structure-drools</artifactId>
</dependency>
```
